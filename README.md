# 数字听力训练（Vue）

基于 **Vue 3 + TypeScript + Vite** 的数字听力练习应用，包含三个模块：

1. 电话号码听力：随机 30 题，支持自定义号码位数。
2. 日期听力：随机 30 题，支持自定义日期范围。
3. 数字听力：随机 30 题，支持配置小数点前后位数。

应用使用浏览器的 **Web Speech API** 进行美式英语朗读，可从可用的 `en-US` 声音中选择（最多展示 5 个），并提供「重复发音 / 下一个 / 显示答案」操作。

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```
